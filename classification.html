Weapon Type Classification

This section details the classification of weapon types in the Homicide Reports dataset (1980-2014) using K-Nearest Neighbors (KNN), Naïve Bayes, and Random Forest algorithms. The dataset was preprocessed by encoding categorical variables (e.g., victim sex, relationship) using LabelEncoder and scaling numerical features (e.g., victim age) to ensure model compatibility. The models classify homicides into four weapon types: Firearm, Knife, Blunt Object, and Unknown.
